## Description
Partytown implementation in GatsbyJS as a plugin

## How to install

Optional: Add this to your .env file, if you won't add it in the config.
GTM_ID=

npm i gatsby-plugin-partytown

## gatsby-config.js
{
      resolve: `gatsby-plugin-partytown`,
      options: {
        gtm_id: "YOUR-GTM-ID-HERE",
      },
    },
